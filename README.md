# 🚀 30 Desafios de JavaScript

Este repositório contém a resolução de **30 desafios de lógica de programação em JavaScript**, organizados por nível de dificuldade.

O objetivo é praticar e evoluir habilidades fundamentais de programação, desde conceitos básicos até problemas mais avançados.

---

## 🎯 Objetivos

* Fortalecer a lógica de programação
* Praticar JavaScript na prática
* Melhorar a resolução de problemas
* Desenvolver pensamento algorítmico
* Construir um portfólio sólido

---

## 📂 Estrutura do projeto

Os desafios estão organizados em cinco níveis:

* 🟢 [Very Easy](./very-easy) → conceitos básicos
* 🟡 [Easy](./easy) → lógica intermediária
* 🟠 [Medium](./medium) → desafios moderados
* 🔴 [Hard](./hard) → problemas mais complexos
* ⚫ [Very Hard](./very-hard) → desafios avançados

Cada desafio possui:

* um arquivo `index.js` com a solução
* um `README.md` com explicação do problema e da abordagem
