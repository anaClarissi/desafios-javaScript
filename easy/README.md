# 🟡 Easy - Desafios de JavaScript

Esta pasta contém desafios classificados como **Easy**, focados em consolidar a lógica de programação e introduzir problemas um pouco mais elaborados utilizando JavaScript.

O objetivo desses exercícios é desenvolver habilidades como:

* resolução de problemas
* manipulação de arrays e strings
* uso de estruturas de repetição
* aplicação de funções
* raciocínio lógico intermediário

Esses desafios servem como uma transição entre o nível básico e problemas mais complexos.
