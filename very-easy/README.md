# 🟢 Very Easy - Desafios de JavaScript

Esta pasta contém desafios classificados como **Very Easy**, focados em lógica básica de programação utilizando JavaScript.

O objetivo desses exercícios é fortalecer os fundamentos essenciais, como:

* manipulação de variáveis
* estruturas condicionais
* loops
* funções
* operações matemáticas simples

Esses desafios são ideais para iniciantes e para aquecer antes de problemas mais complexos.
